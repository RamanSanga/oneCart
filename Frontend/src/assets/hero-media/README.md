# Hero Media Folder

Drop your uploaded hero media files here.

Supported:
- 1 video file: `.mp4`, `.webm`, or `.mov`
- Up to 4 image files: `.png`, `.jpg`, `.jpeg`, `.webp`, or `.avif`

Behavior:
- If a video is present, the background uses that single video.
- If no video is present but images exist, the background uses 4 photo slots.
- If this folder is empty, the app falls back to the existing built-in background media.
